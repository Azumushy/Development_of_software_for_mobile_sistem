# Архитектура приложения

## Принципы разработки компонентов

Все компоненты следуют единой структуре:

1. **Импорты основных библиотек** (React, React Native)
2. **Импорты своих компонентов**
3. **Импорты утилит и стилей**
4. **Код компонента**
5. **Стили компонента**
6. **Экспорт**

## Пример структуры компонента

```javascript
// 1. Импорты основных библиотек
import React from 'react';
import { View, Text, StyleSheet } from 'react-native';

// 2. Импорты своих компонентов
import Button from './Button';
import Input from './Input';

// 3. Импорты утилит и стилей
import { COLORS } from '../utils/constants';

// 4. Код компонента
const MyComponent = ({ prop1, prop2 }) => {
  return (
    <View style={styles.container}>
      {/* ... */}
    </View>
  );
};

// 5. Стили компонента
const styles = StyleSheet.create({
  container: {
    // ...
  },
});

// 6. Экспорт
export default MyComponent;
```

## Принципы проектирования

1. **Единственная ответственность**: Каждый компонент решает только одну задачу
2. **Разбиение больших компонентов**: Если компонент становится слишком большим, он разбивается на подкомпоненты
3. **Независимость от экранов**: Компоненты не зависят от конкретного экрана
4. **Использование props**: Все настройки передаются через props
5. **Избегание жестко закодированных значений**: Все константы вынесены в `utils/constants.js`

## Компоненты

### Базовые UI компоненты
- `Button` - переиспользуемая кнопка с вариантами стилей
- `Input` - поле ввода с валидацией и поддержкой ошибок
- `StarRating` - компонент рейтинга со звездами

### Компоненты домена
- `DoctorCard` - карточка врача для списка
- `SearchBar` - строка поиска
- `FilterBar` - панель фильтров и сортировки
- `ReviewCard` - карточка отзыва
- `ReviewList` - список отзывов

## Контекст

- `AuthContext` - управление аутентификацией и состоянием пользователя

## Утилиты

- `storage.js` - работа с AsyncStorage
- `validation.js` - функции валидации форм
- `constants.js` - константы приложения (цвета, опции сортировки, специальности)

## Навигация

- `AppNavigator` - главный навигатор приложения
  - Стек навигация для экранов
  - Табы для главного экрана и профиля
  - Условная навигация в зависимости от состояния аутентификации
