# Инструкция по настройке

## Настройка иконок и изображений

Приложение требует следующие файлы изображений в папке `assets/`:

1. `icon.png` - иконка приложения (1024x1024)
2. `splash.png` - экран загрузки (1242x2436)
3. `adaptive-icon.png` - адаптивная иконка для Android (1024x1024)
4. `favicon.png` - иконка для веб-версии (48x48)

Вы можете:
- Создать папку `assets/` в корне проекта
- Добавить туда необходимые изображения
- Или временно закомментировать ссылки на изображения в `app.json`

## Первый запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. При первом запуске будут созданы тестовые данные о врачах автоматически.

## Структура данных

Данные хранятся локально в AsyncStorage:
- Врачи: ключ `doctors`
- Отзывы: ключ `reviews`
- Пользователи: ключ `user` и `user_{id}_password`

## Тестовые данные

При первом запуске создаются 5 тестовых врачей:
- Терапевт (15 лет опыта)
- Кардиолог (12 лет опыта)
- Невролог (8 лет опыта)
- Педиатр (20 лет опыта)
- Хирург (18 лет опыта)
